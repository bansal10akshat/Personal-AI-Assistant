# Personal AI Document Assistant

## Features
- Upload multiple PDFs
- Ask questions using natural language
- Answers with source context
- Intent detection
- Chat history + export
- Clear documents functionality

## Tech Stack
- Streamlit
- FAISS (Vector DB)
- HuggingFace Embeddings
- Transformers (QA Model)

## Run Locally
pip install -r requirements.txt
streamlit run app.py
